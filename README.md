# 🚀 FinancialGuard AI  
### Intelligent AI-Powered Financial Planning & Investment Assistant

FinancialGuard AI is a comprehensive fintech application that helps users analyze their finances, plan investments, and make smarter financial decisions using AI.

It combines financial modeling, AI advisory, real-time stock insights, and interactive tools into a single unified platform.

---

## 📌 Features

### 🧠 AI Financial Analysis
- Analyzes income, expenses, and investments  
- Calculates savings, risk level, and financial health  
- Generates long-term wealth projections  

---

### 🤖 AI Financial Advisor
- Uses Gemini AI for personalized financial advice  
- Suggests:
  - Expense optimization  
  - Investment strategies  
  - Risk management  

---

### 📊 AI Financial Planner
- Automatically distributes savings into:
  - Stocks  
  - Mutual Funds  
  - ETFs  
  - Crypto  
  - Gold  
  - REITs  
- Based on user risk profile  

---

### 📈 Wealth Growth Simulation
- Visual graph of projected financial growth  
- Shows long-term investment impact  

---

### 📉 Stock Insight System
- Search stock (AAPL, TSLA, etc.)  
- Displays:
  - Current price  
  - Trend graph  
- Powered by Alpha Vantage API  

---

### 🧮 Finance Calculators
- SIP Calculator  
- Mutual Fund Calculator  
- Graph-based return visualization  

---

### 💬 Savaya AI Chatbot
- AI financial assistant  
- Answers questions like:
  - “Where should I invest ₹10,000?”  
- Gives simple financial explanations  

---

## 🛠️ Tech Stack

- Flutter (Dart)  
- Gemini AI API  
- Alpha Vantage API  
- fl_chart (Graph Visualization)
